# Introduction

Welcome to the GEMINI Breeding project. 

GEMINI is a platform for automated field crop phenotyping, enabling researchers to collect, process, and analyze massive amounts of high-throughput field imagery.

## Project Overview
Learn more about our research and the full project scope at our official website:
[**https://projectgemini.ucdavis.edu/**](https://projectgemini.ucdavis.edu/)

# User Guide

Welcome to the GEMINI Breeding application.

## Installation
To get started with the GEMINI application, download the latest executable for your operating system from our official GitHub Releases page:

[**https://github.com/GEMINI-Breeding/GEMINI-App/releases**](https://github.com/GEMINI-Breeding/GEMINI-App/releases)

## System Requirements
- **Docker Desktop:** Required for certain data processing features in the application. Ensure it is running when performing these tasks.
